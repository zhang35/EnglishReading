# 英语学习记录

每日阅读一到两篇英文文章（扇贝阅读等），整理成可直接发布到微信公众号的学习笔记。

## 目录结构

| 路径 | 说明 |
|------|------|
| `new/` | 待处理：每天放入 1-2 张文章截图（JPG）+ 对应的 1-2 段朗读音频（m4a） |
| `archive/` | 已处理：按文章英文标题命名的图片与音频 |
| `learning_notes/` | 学习笔记：按日期命名的纯文本合订本 |
| `TASK.md` | 处理流程与学习笔记格式规范 |

学习笔记文件命名：`learning_notes/YYYY-MM-DD 学习笔记.txt`

## 日常使用

1. 将当天的截图和录音放进 `new/`
2. 按 `TASK.md` 中的步骤处理（或由助手执行 `@TASK.md`）
3. 打开 `learning_notes/` 中对应日期的笔记（可能是单篇或双篇），复制到微信公众号后台发布

## 学习笔记内容

面向雅思约 6.5 水平的中文英语学习者，每篇文章包含：

- **文章标题** — 英文标题与中文副标题各加一枚与主题相关的 emoji（如 🕶️ ☕）
- **✨ 概述** — 中文概括正文；若文章是列举型（如「7 种食物」「6 个方法」），用 `•` 分条列出，否则用简短段落
- **💎 词汇** — 重点词 + 美音音标 + 词性 + 中文释义
- **🎤 地道表达** — 短语/句式 + 释义 + 例句与译文（每组之间空一行）

详细排版要求见 [TASK.md](./TASK.md)。
